Como Usar no Seu Post

Simplesmente substitua as imagens do seu HTML pelas classes apropriadas:

<!-- Imagem principal -->
<img src="../images/desenvolvedor-codigo-limpo.jpg" 
     alt="Desenvolvedor escrevendo código limpo" 
     class="featured-image" 
     loading="lazy">

<!-- Imagem no conteúdo -->
<img src="../images/top-10-boas-praticas.jpg" 
     alt="Infográfico com as 10 práticas" 
     class="content-image" 
     loading="lazy">

<!-- Imagem com legenda -->
<div class="image-container">
    <img src="../images/beneficios-boas-praticas.jpg" 
         alt="Gráfico dos benefícios" 
         loading="lazy">
    <p class="image-caption">Gráfico mostrando os benefícios das boas práticas</p>
</div>
